# Train the model (TrainFlexiUnet)

## Goal
Train a UNet to predict three probability maps per image:
- Foreground (fg)
- Boundary
- Seed (centerline)

## Required inputs
- Training root folder (`--root`) generated by CreateDataSet

## Outputs
- Model weights: `.pth`
- Metrics log: `.csv`
